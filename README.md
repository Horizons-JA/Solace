## Installation

### Prequsites

- NodeJS
- Expo App (iOS & Android)

### Clone the repository

```bash
git clone https://github.com/Horizons-JA/Solace.git
```

### Run the project in Expo

```bash
npx expo start
```

Follow the steps given in the terminal.
